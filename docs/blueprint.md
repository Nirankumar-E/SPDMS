# **App Name**: TN-PDS Portal Replica

## Core Features:

- Header Section: Dark top bar with helpline numbers, language switch, and accessibility icons.
- Left Sidebar: Vertical navigation with colored buttons and icons for various services.
- Center Section: Dashboard displaying key statistics in grid format with animated numeric counters.
- Right Sidebar: Color-coded panels providing citizen-centric services with clickable elements.
- Middle Banners: Large clickable banners for complaint registration and status tracking.
- Helpline Section: Full-width colored strip with helpline numbers and WhatsApp support.
- Footer Section: Four-column footer with acts, policies, quick links, related departments, and contact details.
- Grievance Submissions: Store beneficiary grievance submissions using Firestore.
- Beneficiary Data: Store beneficiary information using Firestore.

## Style Guidelines:

- Primary color: Forest green (#228B22) to convey a sense of government authority and reliability.
- Background color: Light grayish-green (#F0FAF0) to create a calm and official backdrop.
- Accent color: Maroon (#800000) for buttons and interactive elements to add emphasis.
- Body and headline font: 'PT Sans', a humanist sans-serif, for clear readability in both Tamil and English.
- Use simple, government-style icons to represent services and categories.
- Three-column responsive layout with a prominent header, sidebars, and footer.
- Subtle transitions and hover effects to enhance user interaction.
